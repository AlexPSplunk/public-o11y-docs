********************************************************************
Instrument Rust applications for Splunk Observability Cloud
********************************************************************

.. meta:: 
    :description: Get started with instrumenting your Rust application for Splunk Observability Cloud.

.. toctree::
    :hidden:

    Requirements <requirements>
    Instrument your Rust application <instrument-rust>

You can send application traces and metrics from your Rust applications to Splunk Observability Cloud using the OpenTelemetry instrumentation for Rust.

To instrument your Rust applications, follow these steps:

#. Make sure you've met the prerequisites. See :ref:`rust-requirements`
#. Instrument your application. See :ref:`instrument-rust`
