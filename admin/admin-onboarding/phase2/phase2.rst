In the pilot phase your focus is to onboarding teams. These teams represent use cases that can be used to show the power and benefit of Splunk Observability Cloud to the rest of the organization. Your goal in the pilot phase is to roll out initial pilots of Splunk Infrastructure Monitoring and Splunk Application Performance monitoring. The steps to roll out these products are described in more detail in the following topics:
