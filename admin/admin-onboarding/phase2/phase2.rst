Admin onboarding guide phase 2: Pilot rollout
****************************************************************

.. toctree::

In the pilot rollout phase your goal is to onboarding teams. These teams represent use cases that show the power and benefit of Splunk Observability Cloud to the rest of the organization. Your focus in the pilot rollout phase is to roll out initial pilots of Splunk Infrastructure Monitoring and Splunk Application Performance monitoring. There are 3 high-level sets of activities described in the following topics:

* :ref:`Pilot rollout part 1: Plan your rollout <phase2-rollout-plan>`
* :ref:`Pilot rollout part 2: Initial pilot rollout for Splunk Infrastructure Monitoring <phase2-im>`.
* :ref:`Pilot rollout part 3: Initial pilot rollout for Splunk Application Performance Monitoring <phase2-apm>`.
